# Çift Yarık Deneyi Simülasyonu (Double Slit Experiment Simulation)

Bu proje, Python kullanılarak Schrödinger Denklemi temelinde geliştirilen bir **çift yarık deneyi simülasyonu** içermektedir. Projenin amacı, kuantum mekaniğinin temel fenomenlerinden biri olan girişim desenini sayısal yöntemlerle görselleştirmektir.

## Özellikler

- 2 boyutlu zaman-bağımlı Schrödinger Denklemi çözümü
- Çift yarık için olasılık yoğunluğu hesaplaması
- Animasyon ile girişim desenlerinin dinamik gösterimi
- NumPy ve Matplotlib kullanılarak yüksek performanslı simülasyon
